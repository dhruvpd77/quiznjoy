Upload Questions from Excel

Upload an Excel file with questions for a specific subject

📋 Excel File Format (works for all subjects)

Your Excel file should contain the following columns:

  • unit_number or unit_numb – Unit number (1, 2, 3…10)
  • question_text – The question text (supports newlines and indentation)
  • MCQ Answer – Correct answer (A, B, C, or D)
  • A, B, C, D or option A / option B / option C / option D – Option texts (all 4 required for MCQs)
  • Added By – (Optional)
  • Verified By – (Optional)

If your header row is in the second row, the upload will use row 2 as the header.

⚠️ Only rows with all 4 options (A–D) filled are imported as MCQs. Rows with missing options are saved as programming questions.

🖼️ Digital Electronics / any subject with images: Embed images in the question column (Insert → Picture in Excel). Images are auto-extracted and attached to that row. You can also add or change images later via Admin → Manage Questions → Edit.

📝 Full Stack Development / code with indent: Use Alt+Enter in a cell for new lines; use spaces for indentation. Indentation is preserved so code displays correctly in quizzes.

∑ Maths formulas: Use \( ... \) for inline math and \[ ... \] for display math (LaTeX). Example: \( x^2 + y^2 = z^2 \)
